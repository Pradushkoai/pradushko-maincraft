# Архитектура Pradushko Maincraft

## Обзор

Игра построена на **Electron + Three.js**. Electron запускает браузерное окно с игрой, Three.js рендерит 3D-сцену.

## Структура файлов

```
pradushko-maincraft/
├── main.js                 ← Electron main process (запуск окна)
├── package.json            ← зависимости и конфиг сборки
├── src/
│   ├── index.html          ← HTML/CSS UI
│   ├── game.js             ← вся игровая логика (один файл)
│   ├── blocks.js           ← определения блоков (документация)
│   ├── items.js            ← предметы, оружие, броня
│   ├── biomes.js           ← биомы
│   ├── mobs.js             ← мобы и боссы
│   ├── recipes.js          ← рецепты крафта
│   ├── structures.js       ← генерация структур
│   └── world.js            ← чанковая система мира
├── assets/                 ← текстуры, звуки
├── worlds/                 ← сохранения (.gitignored)
└── .github/workflows/      ← CI/CD
```

## Чанковая система мира

Мир 256×256×64 блоков делится на чанки 16×16×64. Загружаются только чанки в радиусе `RENDER_DISTANCE` (4 чанка = 64 блока) вокруг игрока.

```

[ ] [ ] [ ] [ ] [ ] [ ] [ ] [ ] [ ]
[ ] [ ] [ ] [ ] [ ] [ ] [ ] [ ] [ ]
[ ] [ ] [ ] [P] [P] [P] [P] [P] [ ]     P = загруженные чанки
[ ] [ ] [ ] [P] [P] [P] [P] [P] [ ]     вокруг игрока
[ ] [ ] [ ] [P] [P] [P] [P] [P] [ ]
[ ] [ ] [ ] [P] [P] [P] [P] [P] [ ]

```

При движении игрока в новый чанк — старые дальние выгружаются, новые ближние генерируются.

## Биомы (9 типов)

Каждый биом определяет:
- Блок поверхности (трава/песок/снег/камень)
- Высоту рельефа
- Тип деревьев
- Список мобов
- Босса
- Структуры

```
PLAINS    → коровы, овцы, свиньи, лошади → Лесной Голем
FOREST    → волки, медведи, лисы, олени   → Лесной Голем
SWAMP     → ведьмы, лягушки, змеи, слизни → Болотная Химера
DESERT    → скорпионы, мумии, верблюды    → Песчаный Червь
SNOW      → белые медведи, йети, пингвины → Ледяной Король
JUNGLE    → ягуары, пантеры, обезьяны     → Король Ягуаров
MOUNTAINS → орлы, тролли, гарпии          → Каменный Титан
OCEAN     → акулы, кальмары, дельфины     → Морской Змей
SAVANNA   → львы, жирафы, слоны           → Царь Львов
```

## Раритеты

5 уровней редкости (цвет рамки предмета):
- ⚪ COMMON (белый) — базовые материалы
- 🟢 UNCOMMON (зелёный) — улучшенные
- 🔵 RARE (синий) — редкие
- 🟣 MYTHIC (фиолетовый) — мифические
- 🟡 LEGENDARY (золотой) — легендарные

## Рендеринг

Используется `InstancedMesh` — один mesh на тип блока, содержащий все видимые блоки этого типа. Только блоки с открытой гранью (сосед — воздух или прозрачный) попадают в mesh.

## Сохранение мира (планируется)

- `worlds/save1/chunks/` — бинарные файлы чанков
- `worlds/save1/player.json` — позиция, инвентарь, экипировка
- `worlds/save1/structures.json` — размещённые структуры

При выходе — сериализация. При входе — десериализация и дозагрузка чанков.
