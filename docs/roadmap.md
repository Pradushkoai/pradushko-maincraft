# Roadmap — план разработки

## ✅ v1.0 — Базовая игра (готово)

- [x] Чанковая система мира 256×256×64
- [x] 9 биомов с уникальными параметрами
- [x] 90+ типов блоков
- [x] 60+ предметов (материалы, еда, оружие, броня)
- [x] 100+ рецептов крафта
- [x] 50+ мобов (мирные и враждебные)
- [x] 8 боссов (по одному на биом)
- [x] 5 раритетов (Common → Legendary)
- [x] Дома с интерьером и крафтовыми столами
- [x] Подземелья с этажами
- [x] Структуры: пирамиды, храмы, кораблекрушения, ведьмины хижины, ледяные замки
- [x] Миникарта с крупными отметками
- [x] Большая карта (M) с легендой
- [x] Инвентарь с крафтовым столом 3×3
- [x] Боевка с уроном, отбрасыванием, критами
- [x] Система голода и здоровья
- [x] Мана для магических посохов
