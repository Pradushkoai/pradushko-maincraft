# Pradushko Maincraft 🎮

3D-песочница в стиле Minecraft, написанная на **Electron + Three.js**.

Большой мир с биомами, уникальными мобами, боссами, системой раритетов оружия и брони, крафтом мебели и подземельями.

## 🚀 Установка для разработки

```bash
# Установить Node.js 18+ с сайта https://nodejs.org
# Затем в папке проекта:
npm install
npm start          # запустить игру в режиме разработки
```

## 📦 Сборка Windows .exe

```bash
npm run dist       # собрать .exe в папке dist/
```

Готовый файл: `dist/PradushkoMaincraft-Setup-1.0.0.exe`

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| **WASD** | Движение |
| **Пробел** | Прыжок |
| **Shift** | Спуск в полёте |
| **Мышь** | Обзор (клик — захват) |
| **ЛКМ** | Копать / бить оружием |
| **ПКМ** | Поставить блок / использовать |
| **1-9** | Выбор слота хотбара |
| **Колесо** | Листать хотбар |
| **E** | Инвентарь + крафт |
| **M** | Большая карта |
| **F** | Полёт |
| **R** | Подобрать лут |
| **ESC** | Пауза |

## 🌍 Биомы

- 🌲 **Лес** — густые деревья, волки, медведи, босс: Лесной Голем
- 🌫️ **Болото** — туман, лужи, ведьмы, босс: Болотная Химера
- 🏜️ **Пустыня** — песок, кактусы, скорпионы, босс: Песчаный Червь
- ❄️ **Снежные земли** — снег, ели, белые медведи, босс: Ледяной Король
- 🌴 **Джунгли** — густая растительность, попугаи, босс: Ягуар
- ⛰️ **Горы** — камень, орлы, босс: Каменный Титан
- 🌊 **Океан** — вода, рыба, босс: Морской Змей

## ⚔️ Раритеты

- ⚪ **Обычное** (Common) — базовые материалы
- 🟢 **Улучшенное** (Uncommon) — улучшенная защита/урон
- 🔵 **Редкое** (Rare) — особые эффекты
- 🟣 **Мифическое** (Mythic) — мощные бонусы
- 🟡 **Легендарное** (Legendary) — лучшее снаряжение

## 📁 Структура проекта

```
pradushko-maincraft/

│   ├── biomes.js          ← определения биомов

│   └── ui.js              ← интерфейс

│   ├── textures/          ← текстуры
│   └── sounds/            ← звуки
├── worlds/                ← сохранения (.gitignored)

│   ├── roadmap.md         ← план разработки
│   └── architecture.md    ← архитектура

```
